Калькулятор широкополосных сетей для проекта Giga предназначен для выбора технологий и расчета затрат на подключение школ к Интернету.

Этот инструмент реализует методики расчета локальных компьютерных сетей школы и подключений к сетям широкополосного доступа. Он помогает выбрать оптимальные варианты развертывания сети с точки зрения капитальных и эксплуатационных затрат.

Калькулятор широкополосных сетей для проекта Giga предлагает два режима расчета: для одной школы и для группы школ.

Первый режим может использоваться администрацией школы для определения потребностей своей школы в локальной компьютерной сети и широкополосном соединении, а также для получения справочного отчета с оценкой затрат и возможными сценариями реализации. Он не требует регистрации и может быть использован сразу.

Второй режим предназначен для использования на уровне образовательных или телекоммуникационных органов для планирования охвата ряда школ в конкретном регионе. Это позволяет создавать несколько проектов и производить расчеты по количеству школ с учетом дополнительной информации об их взаимном расположении.
